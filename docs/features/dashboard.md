# All-in-One Dashboard

Manage all your customer communication from a single, intuitive dashboard.

## Dashboard Highlights

- **Order Search**: Quickly find orders by product, status, or customer.
- **Bulk Actions**: Select multiple orders and send updates in one go.
- **Status Overview**: See fulfillment and communication status at a glance.
- **Easy Navigation**: Access filters, templates, and support in one place.
